Premium illuminated signage that commands attention.

Reverse acrylic lit signs combine a stainless steel face with high-efficiency LED illumination through polycarbonate acrylic backing. Your design is laser-cut directly into the steel face — letters, logos, and shapes become glowing cutouts that read brilliantly day and night.

The cool-white LED modules (10000–12000K) are rated IP67 for full outdoor weatherproofing, and the Class 2 transformer meets UL standards for exterior sign installations. Every sign ships ready to install with a wiring diagram, installation pattern, and mounting template.

  • 0.4" polycarbonate translucent acrylic backing
  • Mynice IP67 LED strip modules — 160° beam, 0.7W & 1.2W
  • 12V/5A 60W Class 2 transformer with 3-ft power cord
  • UL Listed sign section labels for outdoor installation
  • Wiring diagram, installation pattern, and mounting template

Vector file setup guide

This product requires true vector artwork. Raster images (JPEG, PNG, PSD) cannot be used for laser cutting.

Accepted formats: PDF, EPS, or AI only

JPEG, PNG, PSD, and Word documents are not accepted. The laser cutter requires vector paths — not pixel data. If you only have a raster logo, request our design help service to redraw it as a vector.

Convert all text to outlines

In Illustrator: Select all text → Type → Create Outlines. In Inkscape: Path → Object to Path. Without outlined text, fonts will substitute or fail at the cutter.

Scale to final dimensions

Set your artboard to the exact sign dimensions you entered above. The file is cut at the dimensions it arrives — do not use scale percentages or proxy sizes.

Minimum stroke / letter width: 1 inch

Any letter stroke, line, or cutout narrower than 1 inch will be too fragile to hold in the steel face after cutting. Thin strokes may be adjusted or flagged before production.

No bleed or crop marks required

Unlike print products, channel letters have no bleed. Your design paths define the cut lines exactly. Keep your paths clean and closed.

Vector checklist

  • PDF, EPS, or AI format
  • All text converted to paths
  • Scaled to final sign size
  • Min 1" stroke/letter width
  • No embedded raster images
  • Closed paths for all cutouts
  • Pantone code noted if custom
